{{< alert title="《中华人民共和国国旗法》" color="info" >}}
中华人民共和国国旗是五星红旗。<br>
中华人民共和国国旗是中华人民共和国的象征和标志。每个公民和组织，都应当尊重和爱护国旗。<br>
{{< /alert >}}

{{< alert title="历史沿革" color="success" >}}
1949年7月4日，新政治协商会议筹备会第六小组决定发出征集国旗、国徽图案和国歌词谱的启事。<br>
1949年9月27日中国人民政治协商会议第一届全体会议通过中华人民共和国国旗为五星红旗。<br>
1954年9月第一届全国人民代表大会通过第一部《中华人民共和国宪法》以后至今，五星红旗一直是中华人民共和国国旗。<br>
2020年10月17日，第十三届全国人民代表大会常务委员会第二十二次会议通过了关于修改《中华人民共和国国旗法》的决定，并于2021年1月1日起正式施行。<br>
{{< /alert >}}

{{< alert title="《中华人民共和国国歌法》" color="info" >}}
中华人民共和国国歌是《义勇军进行曲》。<br>
中华人民共和国国歌是中华人民共和国的象征和标志。<br>
一切公民和组织都应当尊重国歌，维护国歌的尊严。<br>
{{< /alert >}}

{{< alert title="历史沿革" color="success" >}}
《义勇军进行曲》是1935年5月在中国上海上映的华语电影《风云儿女》的主题歌，田汉作词、聂耳作曲（以及阿隆·阿甫夏洛穆夫编写管弦乐总谱）。<br>
1949年9月27日定为中华人民共和国暂定国歌。<br>
1968年-1982年，国歌歌词随即被弃用而改为只奏纯音乐或集体填词版歌词（《继续革命的战歌》）。<br>
1982年撤销1978年的新词，恢复田汉作词版。<br>
2004年3月14日，第十届全国人民代表大会第二次会议通过宪法修正案，正式将《义勇军进行曲》作为国歌写入宪法。<br>
2017年9月1日，第十二届全国人民代表大会常务委员会第二十九次会议通过了《中华人民共和国国歌法》，并于2017年10月1日起施行。<br>
2024年9月1日，国务院新闻办举行新闻发布会，发布国歌标准演奏曲谱和官方录音版本。<br>
{{< /alert >}}

{{< alert title="《中华人民共和国国徽法》" color="info" >}}
中华人民共和国国徽，中间是五星照耀下的天安门，周围是谷穗和齿轮。<br>
中华人民共和国国徽是中华人民共和国的象征和标志。<br>
一切组织和公民，都应当尊重和爱护国徽。
{{< /alert >}}

{{< alert title="历史沿革" color="success" >}}
1950年6月23日中国人民政治协商会议第一届全国委员会第二次会议提出，
同年6月28日中央人民政府委员会第八次会议通过，
同年9月20日《中央人民政府公布中华人民共和国国徽的命令》公布。<br>
2020年10月17日，第十三届全国人民代表大会常务委员会第二十二次会议通过了关于修改《中华人民共和国国徽法》的决定，并于2021年1月1日起正式施行。<br>
{{< /alert >}}
